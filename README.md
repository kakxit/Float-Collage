# Float Collage (拼好图)

一款沉浸式自由拼图工具，让你在桌面上自由组合和排列图片。

![Float Collage](freepn-export.png)

## ✨ 特性

- 🖼️ **自由拼图** - 在无限画布上自由拖拽、缩放、旋转图片
- 📐 **参考线系统** - 智能吸附和临时对齐线，精确排列图片
- 🎯 **图层管理** - 完整的图层控制，支持锁定、隐藏、排序
- 🔄 **比例锁定** - 支持多种画布比例，保持构图完美
- 👻 **鼠标穿透** - 开启后可以点击穿透到桌面下方的窗口
- 📌 **始终置顶** - 让拼图窗口始终显示在最前
- 🎨 **透明度调节** - 调整画布背景透明度，与桌面融为一体
- ⚡ **快捷键支持** - 丰富的键盘快捷键，提升操作效率
- 🌍 **多语言** - 支持中文界面

## 🚀 快速开始

### 下载安装

前往 [Releases](https://github.com/kakxit/Float-Collage/releases) 页面下载最新版本。

### 开发环境

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建应用
npm run build

# 打包应用
npm run dist
```

## 📖 使用指南

### 导入图片

- 点击工具栏的"导入"按钮选择图片
- 直接拖拽图片文件到窗口
- 使用快捷键 `Ctrl+O` 打开文件选择器
- 使用快捷键 `Ctrl+V` 粘贴剪贴板中的图片

### 操作图片

- **移动** - 拖拽图片
- **缩放** - 拖拽图片四角
- **旋转** - 拖拽图片上方的旋转控制点
- **删除** - 选中图片后按 `Delete` 键

### 画布操作

- **平移画布** - 按住 `Space` 键并拖拽
- **调节透明度** - 按住 `Alt` 键并滚动鼠标滚轮
- **临时关闭吸附** - 按住 `Alt` 键并拖拽图片

### 参考线功能

1. 点击工具栏的"参考线"按钮
2. 设置横向和竖向参考线数量
3. 点击"生成"创建参考线
4. 开启"智能吸附"让图片自动对齐参考线
5. 开启"临时对齐线"在拖拽时显示对齐提示

### 图层管理

1. 点击工具栏的"图层"按钮
2. 查看所有图片图层
3. 点击眼睛图标切换可见性
4. 点击锁图标锁定图层
5. 使用上下箭头调整图层顺序

### 画布比例

1. 点击工具栏的"比例"按钮
2. 选择预设比例（1:1、4:3、16:9 等）
3. 或输入自定义比例
4. 选择窗口缩放策略：
   - **画面随窗口缩放** - 内容随窗口大小缩放
   - **保持比例贴边** - 保持画布比例，内容贴边显示

### 特殊功能

- **鼠标穿透** - 开启后可以点击穿透到下方窗口，鼠标移到工具栏时自动恢复交互
- **始终置顶** - 让窗口始终显示在其他窗口之上
- **透明度调节** - 调整画布背景透明度，与桌面背景融合

### 导出图片

1. 点击工具栏的"导出"按钮
2. 选择导出格式（PNG、JPEG、SVG）
3. 调整图片质量
4. 点击"导出图片"保存

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+O` | 导入图片 |
| `Ctrl+V` | 粘贴图片 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |
| `Delete` | 删除选中图片 |
| `Space + 拖拽` | 平移画布 |
| `Alt + 滚轮` | 调节透明度 |
| `Alt + 拖拽` | 临时关闭吸附 |

## 🛠️ 技术栈

- **Electron** - 跨平台桌面应用框架
- **React** - UI 框架
- **TypeScript** - 类型安全
- **Fabric.js** - 画布渲染引擎
- **Zustand** - 状态管理
- **i18next** - 国际化支持
